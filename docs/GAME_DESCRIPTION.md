
# Описание игры "Алфавит"

**"Алфавит"** — это современная интерактивная браузерная игра, предназначенная для проведения командных соревнований и викторин. Игра сочетает в себе классическую механику тренировки памяти (Memory) с элементами интеллектуального квиза.

## Основная концепция
Игроки (команды) по очереди открывают карточки с буквами алфавита на большом игровом поле. Цель — найти пару одинаковых букв. При нахождении совпадения команда получает баллы, а если включен режим викторины — получает возможность ответить на бонусный вопрос для заработка дополнительных очков.

## Ключевые возможности

### 1. Механика "Memory"
*   Красочное игровое поле с полным набором букв алфавита (поддержка русского и английского языков).
*   Плавные 3D-анимации переворота карточек.
*   Автоматический учет найденных пар и визуализация прогресса.

### 2. Интегрированная викторина
*   Три уровня сложности вопросов: легкий, средний и сложный.
*   Бонусные баллы в зависимости от выбранной сложности.
*   Система выбора сложности после нахождения каждой пары.

### 3. Искусственный интеллект (AI Generator)
*   Встроенный генератор вопросов на базе современных моделей (Google Gemini, Qwen, Llama).
*   Возможность мгновенного создания тематических наборов вопросов (например, "История", "Наука", "Библия").
*   Поддержка генерации до 200 уникальных вопросов за один запрос.

### 4. Библиотека наборов (Question Library)
*   Предустановленные (системные) наборы вопросов на различные темы.
*   Возможность импорта и экспорта собственных наборов в формате JSON.
*   Редактор вопросов для ручной настройки контента.

### 5. Командный режим и управление
*   Поддержка от 2 до 5 команд одновременно.
*   Настройка имен команд и лимитов времени на ход.
*   Удобная панель управления ("Home" кнопка, пауза, сброс прогресса).

### 6. Оптимизация для мероприятий
*   Адаптивный интерфейс, который отлично смотрится на проекторах и больших ТВ.
*   Визуальные таймеры для контроля времени хода.
*   Звуковое сопровождение игровых событий (совпадения, ошибки, победа).

## Технические особенности
*   **Сохранение прогресса**: Состояние игры сохраняется локально — можно случайно закрыть браузер и продолжить с того же места.
*   **Мультиязычность**: Полная поддержка интерфейса и игровых механик на русском и английском языках.
*   **Легкость запуска**: Работает в любом современном браузере без установки дополнительных программ.
