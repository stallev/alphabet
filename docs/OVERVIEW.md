
# Обзор проекта "Алфавит" (v2.6)

## Концепция
"Алфавит" — это интерактивное веб-приложение для проведения интеллектуальных викторин, объединяющее механику "Memory" (поиск парных карт) и глубокую библейскую викторину. Проект оптимизирован для работы на больших экранах и проекторах.

## Целевая аудитория
- Молодежные христианские лагеря и конференции.
- Воскресные школы (режим без вопросов).
- Семейные и церковные интеллектуальные вечера.

## Ключевые возможности v2.6
1. **Hybrid AI Engine**: Генерация 200 уникальных вопросов за один запрос через Google Gemini или OpenRouter (Qwen, Llama, DeepSeek).
2. **Global Navigation**: Постоянный доступ к главной странице через "умную" кнопку Home, доступную на всех этапах, кроме стартового.
3. **Advanced State Recovery**: Сохранение прогресса в LocalStorage с автоматической очисткой старых наборов (TTL 72 часа).
4. **Multi-Team Support**: Поддержка от 2 до 5 команд с динамическим переключением хода и визуальным трекингом баллов.
5. **Projector Ready**: Высококонтрастный интерфейс, крупные шрифты и анимации, адаптированные для демонстрации аудитории.
